# Project: Academic Outlier 🎓

## Context
Academic Outlier is a MERN stack application designed to streamline the university discovery and student onboarding process. It features a premium glassmorphic UI, secure authentication (Google OAuth + OTP), and a personalized dashboard for students.

## Mission
To bridge the gap between aspiring students and global academic opportunities through an intuitive, data-driven, and visually stunning interface.

## Tech Stack
- **Frontend**: React.js, Vite, Tailwind CSS, Recharts
- **Backend**: Node.js, Express, MongoDB/Mongoose, Passport.js (Google OAuth), Multer, Nodemailer
- **Infrastructure**: Vercel (Frontend), Render/Koyeb (Backend)

## Core Value Proposition
- **Seamless Discovery**: Access to thousands of global universities.
- **Secure Onboarding**: Multi-step verification and profile management.
- **Premium Experience**: Modern, interactive, and responsive design.

## Success Metrics
- Seamless student conversion from discovery to dashboard.
- High engagement with university filtering tools.
- Secure and reliable document management.
